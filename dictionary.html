Hello, word!

HTML/CSS/JS

Hello, word! is a playful and engaging dictionary application I created as part of my Flatiron School coursework, using HTML, CSS, and JavaScript. The app combines functionality with a touch of humor, offering users word definitions, part of speech, and even audio pronunciations to hear how words are pronounced.

The project integrates a third-party dictionary API to deliver accurate and up-to-date data, while custom error handling ensures a fun experience even when a word isn’t found—complete with witty puns to keep users smiling. Designed with a responsive layout, “Hello, Word” provides a seamless experience across devices, from desktop to mobile.

This project allowed me to refine my skills in API integration, DOM manipulation, and user-centered design while showcasing my creativity and ability to make learning both functional and fun. It reflects my approach to creating accessible, enjoyable, and interactive tools.
