Hello, word!

HTML/CSS/JS

¡Hello, Word! es una aplicación de diccionario divertida y atractiva que creé como parte de mi curso en Flatiron School, utilizando HTML, CSS y JavaScript. La aplicación combina funcionalidad con un toque de humor, ofreciendo a los usuarios definiciones de palabras, categorías gramaticales e incluso pronunciaciones en audio para escuchar cómo se pronuncian las palabras.

El proyecto integra una API de diccionario de terceros para entregar datos precisos y actualizados, mientras que el manejo de errores personalizado asegura una experiencia divertida incluso cuando no se encuentra una palabra, ¡completa con juegos de palabras ingeniosos para mantener a los usuarios sonriendo! Diseñada con un diseño responsivo, “Hello, Word” ofrece una experiencia fluida en todos los dispositivos, desde computadoras de escritorio hasta móviles.

Este proyecto me permitió perfeccionar mis habilidades en integración de API, manipulación del DOM y diseño centrado en el usuario, mientras demostraba mi creatividad y capacidad para hacer que el aprendizaje sea tanto funcional como divertido. Refleja mi enfoque para crear herramientas accesibles, agradables e interactivas.
