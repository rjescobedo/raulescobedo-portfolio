TheTrainer

React

The Trainer es mi primer proyecto importante, iniciado durante las primeras etapas de mi curso de Bootstrap. Este sitio web y aplicación están diseñados para ayudar a los entrenadores personales y sus clientes a conectarse de manera fluida.

Los entrenadores pueden crear y compartir ejercicios y rutinas personalizadas adaptadas a las necesidades de sus clientes. Cada ejercicio incluye videos instructivos e imágenes para demostrar la forma correcta, junto con una breve descripción para orientación. Los clientes pueden registrar sus series, repeticiones y pesos directamente en la aplicación. Una vez que se completa una rutina, los entrenadores son notificados, lo que les permite celebrar el progreso de sus clientes y brindarles motivación continua. En el futuro cercano, planeo incorporar una función de chat para facilitar la comunicación continua sobre las sesiones de entrenamiento, rutinas personalizadas y más.

Uno de los mayores desafíos que enfrenté fue la transición de este proyecto de HTML, CSS y Bootstrap básicos a una aplicación React. A lo largo del proceso, refiné el diseño y la funcionalidad para asegurar una experiencia de usuario intuitiva y agradable. Mirando hacia el futuro, planeo convertir este proyecto a React Native, haciéndolo accesible tanto como un sitio web como una aplicación móvil. Mi objetivo final es ofrecer esta solución a gimnasios y entrenadores personales, empoderándolos para elevar sus servicios.
